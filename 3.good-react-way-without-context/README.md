# Rick & Morty

Приложение состоит из двух страниц:
- Страницы персонажей из Рик и Морти.
- Страница с любимыми персонажами, куда попадают выбранные нажатием на звездочку.

Версия приложения с Zustand как стейт-менеджером.

React.context используется только для выстраивания архитектурных границ и DI.

Использование Zustand позволяет избавится от лишних перерисовок.

Но давайте абстрагироваться от Zustand, дело не в нем. 

Дело в хуке useSyncExternalStore, который используется под капотом у всех известных стейт-менеджеров.
